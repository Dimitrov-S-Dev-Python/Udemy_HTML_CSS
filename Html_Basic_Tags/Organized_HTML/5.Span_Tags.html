Spans - An Inline Element Container

Maybe I want to style THIS SECTION A SPECIFIC WAY! If I style the whole paragraph, the WHOLE paragraph will be stiled the same. Which is not the desired result!

Maybe I want to style

THIS SECTION A SPECIFIC WAY!
If I use a div to target that section it places the section on a new line and takes up the whole of the screen since div is a block level element. This is not what we want!

Maybe I want to style THIS SECTION A SPECIFIC WAY! To do this, I just have to wrap the text in a span.
